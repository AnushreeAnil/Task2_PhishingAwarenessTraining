# 🛡️ PhishGuard - Phishing Awareness Training

**CodeAlpha Cyber Security Internship - Task 2**

An interactive web-based phishing awareness training module designed to educate users about phishing attacks, fake websites, social engineering techniques, and cybersecurity best practices. The module provides engaging learning content, interactive challenges, risk assessment, and quizzes to improve cybersecurity awareness.

---

## 📌 Project Overview

Phishing remains one of the most common cyber threats, targeting individuals and organizations through deceptive emails, fake websites, and social engineering attacks.

This project aims to educate users by providing an interactive learning experience where they can:

- Understand phishing attacks
- Identify phishing emails
- Recognize fake websites
- Learn social engineering tactics
- Complete interactive challenges
- Assess their phishing awareness
- Test their knowledge through quizzes
